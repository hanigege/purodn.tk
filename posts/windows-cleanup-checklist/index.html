Windows 电脑变慢时先检查这 7 件事

电脑卡顿不一定要重装系统，先按这个清单定位常见原因。

电脑变慢时，最稳妥的做法不是先装清理软件，而是确认瓶颈在哪里。下面这份清单适合 Windows 10 和 Windows 11。

1. 看任务管理器

按 Ctrl + Shift + Esc 打开任务管理器，重点看 CPU、内存、磁盘三列。如果某个程序长期占用很高，先记录程序名，再决定是更新、卸载还是限制自启动。

2. 关闭不必要的启动项

进入“设置 > 应用 > 启动”，关闭不常用软件的开机启动。聊天软件、网盘、游戏启动器最常见。

3. 检查磁盘剩余空间

系统盘建议至少保留 20GB 可用空间。空间太低时，Windows 更新、浏览器缓存和临时文件都会拖慢系统。

4. 清理临时文件

进入“设置 > 系统 > 存储 > 临时文件”，勾选确认后再清理。不要随手删除自己不认识的系统目录。

5. 检查浏览器扩展

浏览器扩展过多会明显影响开机后第一轮使用体验。保留密码管理器、广告过滤等必要扩展，停用很久不用的插件。

6. 更新驱动和系统

优先使用 Windows 更新里的驱动；显卡驱动可以从 NVIDIA、AMD 或 Intel 官方工具更新。

7. 判断是否需要升级硬件

如果内存长期接近满载，优先考虑加内存。如果系统装在机械硬盘上，升级 SSD 的体感提升通常最明显。
