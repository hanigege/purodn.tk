Matrix Synapse 机器人：天气推送与服务器监控提醒

记录在自建 Matrix Synapse 上创建 bot 机器人账号，用 Open-Meteo 获取中国上海天气，支持本服用户邀请、聊天命令设置推送时间，并保留管理员服务器监控提醒。

这篇记录一个很实用的小自动化：在自建 Matrix Synapse 上创建一个真正的机器人账号 @bot:666666.xyz。本服用户可以主动邀请它，发送“帮助”查看命令，发送“设置天气时间 07:30”自行设置每日天气推送时间；管理员仍然可以接收磁盘、Docker、TURN / LiveKit 等服务器监控提醒。

这个方案不是 Synapse Server Notices，而是普通 Matrix 机器人账号。好处是体验更像一个联系人：用户可以手动邀请它进私聊，普通用户只使用天气功能，服务器监控提醒则只发给管理员。

天气推送和聊天命令使用 OpenRC 常驻服务控制。天气服务每 20 秒检查一次配置，谁到了自己的推送时间，就给谁发送天气；聊天同步服务每 20 秒处理一次邀请和命令。

rc-service matrix-bot-weather status
rc-service matrix-bot-sync status

服务器监控提醒仍然由 root crontab 控制：

*/5 * * * * /root/data/docker_data/synapse/scripts/run-matrix-bot-monitor.sh

其中 matrix-bot-weather.py 只负责天气内容和到点判断，matrix-bot-sync.py 负责邀请和聊天命令，matrix-bot-monitor.py 每 5 分钟检查服务器异常。天气和监控共用同一个机器人账号，但普通用户只会收到天气，监控提醒只发给监控配置里的管理员。

天气数据使用 Open-Meteo，无需申请 API key。脚本里写的是上海坐标：

latitude: 31.2304
longitude: 121.4737
timezone: Asia/Shanghai

#!/usr/bin/env python3
import datetime as dt
import hashlib
import html
import json
import os
import sys
import time
import urllib.error
import urllib.parse
import urllib.request

CONFIG_PATH = "/root/data/docker_data/synapse/scripts/matrix-bot-weather.json"
WEATHER_CODES = {
    0: "晴", 1: "大部晴朗", 2: "局部多云", 3: "阴", 45: "雾", 48: "雾凇",
    51: "小毛毛雨", 53: "毛毛雨", 55: "较强毛毛雨", 56: "冻毛毛雨", 57: "较强冻毛毛雨",
    61: "小雨", 63: "中雨", 65: "大雨", 66: "冻雨", 67: "强冻雨",
    71: "小雪", 73: "中雪", 75: "大雪", 77: "雪粒",
    80: "小阵雨", 81: "阵雨", 82: "强阵雨", 85: "小阵雪", 86: "强阵雪",
    95: "雷雨", 96: "雷雨伴小冰雹", 99: "雷雨伴强冰雹",
}
HELP_HINT = "发送“天气”可立即查询，发送“设置天气时间 07:30”可修改推送时间。"

def load_json(path, default):
    try:
        with open(path, "r", encoding="utf-8") as f:
            return json.load(f)
    except FileNotFoundError:
        return default

def save_json(path, data, mode=0o600):
    tmp = path + ".tmp"
    with open(tmp, "w", encoding="utf-8") as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
        f.write("\n")
    os.chmod(tmp, mode)
    os.replace(tmp, path)

def request(method, url, token=None, data=None, timeout=30):
    headers = {}
    body = None
    if token:
        headers["Authorization"] = f"Bearer {token}"
    if data is not None:
        body = json.dumps(data, ensure_ascii=False).encode("utf-8")
        headers["Content-Type"] = "application/json"
    req = urllib.request.Request(url, data=body, headers=headers, method=method)
    try:
        with urllib.request.urlopen(req, timeout=timeout) as resp:
            raw = resp.read().decode("utf-8", errors="replace")
            return resp.status, json.loads(raw) if raw else {}
    except urllib.error.HTTPError as e:
        raw = e.read().decode("utf-8", errors="replace")
        try:
            payload = json.loads(raw)
        except Exception:
            payload = {"raw": raw}
        return e.code, payload
    except Exception as e:
        return 0, {"error": str(e)}

def quote(value):
    return urllib.parse.quote(value, safe="")

def token(config):
    with open(config["token_path"], "r", encoding="utf-8") as f:
        return f.read().strip()

def get_weather(config):
    params = urllib.parse.urlencode({
        "latitude": config["latitude"],
        "longitude": config["longitude"],
        "timezone": config.get("timezone", "Asia/Shanghai"),
        "forecast_days": 3,
        "daily": "weather_code,temperature_2m_max,temperature_2m_min,precipitation_probability_max,precipitation_sum,wind_speed_10m_max",
    })
    url = "https://api.open-meteo.com/v1/forecast?" + params
    with urllib.request.urlopen(url, timeout=30) as resp:
        data = json.loads(resp.read().decode("utf-8"))
    daily = data["daily"]
    lines = [f"{config['city']}天气预报", ""]
    today = dt.date.today().isoformat()
    alerts = []
    advice = []
    for i, day in enumerate(daily["time"]):
        label = "今天" if day == today else ("明天" if i == 1 else "后天")
        code = daily["weather_code"][i]
        desc = WEATHER_CODES.get(code, f"天气代码 {code}")
        tmin = daily["temperature_2m_min"][i]
        tmax = daily["temperature_2m_max"][i]
        pop = daily["precipitation_probability_max"][i]
        rain = daily["precipitation_sum"][i]
        wind = daily["wind_speed_10m_max"][i]
        lines.append(
            f"{label} {day}: {desc}，{tmin:.0f}-{tmax:.0f}℃，"
            f"降水概率 {pop}% ，降水量 {rain:.1f} mm，最大风速 {wind:.0f} km/h"
        )
        if i == 0:
            if tmax >= 35:
                alerts.append("高温提醒：今天最高温达到 35℃ 以上，注意防暑补水。")
            if tmin <= 0:
                alerts.append("低温提醒：今天最低温 0℃ 以下，注意防寒防冻。")
            if rain >= 20 or pop >= 70:
                alerts.append("降雨提醒：今天降水概率较高或雨量较大，出门建议带伞。")
            if wind >= 40:
                alerts.append("大风提醒：今天风力偏强，骑车和户外活动注意安全。")
            if code in (95, 96, 99):
                alerts.append("雷雨提醒：今天可能有雷雨，尽量避开强对流时段外出。")
            if tmax >= 30:
                advice.append("穿衣建议：短袖、轻薄透气衣物为主，注意防晒。")
            elif tmin <= 8:
                advice.append("穿衣建议：建议外套或薄棉服，早晚注意保暖。")
            elif tmin <= 15:
                advice.append("穿衣建议：长袖加薄外套比较稳妥。")
            else:
                advice.append("穿衣建议：轻薄长袖或短袖均可，按体感调整。")
            if (tmax - tmin) >= 10:
                advice.append("温差提醒：昼夜温差较大，早晚可多备一件外套。")
            if pop >= 50 or rain > 0:
                advice.append("出行提醒：有降水可能，建议带伞。")
    if alerts:
        lines.extend(["", "天气提醒："] + [f"- {x}" for x in alerts])
    if advice:
        lines.extend(["", "生活建议："] + [f"- {x}" for x in advice])
    lines.extend(["", "数据源：Open-Meteo。", HELP_HINT])
    return "\n".join(lines)

def send_room(config, token_value, room_id, body):
    txn = "weather-" + dt.datetime.now(dt.timezone.utc).strftime("%Y%m%dT%H%M%S")
    txn += "-" + hashlib.sha1((room_id + body).encode("utf-8")).hexdigest()[:10]
    html_body = "<br>".join(html.escape(line) for line in body.splitlines())
    content = {"msgtype": "m.text", "body": body, "format": "org.matrix.custom.html", "formatted_body": html_body}
    url = f"{config['homeserver']}/_matrix/client/v3/rooms/{quote(room_id)}/send/m.room.message/{txn}"
    status, payload = request("PUT", url, token_value, content)
    if status not in (200, 201):
        raise RuntimeError(f"send to {room_id} failed: HTTP {status} {payload}")

def create_dm(config, token_value, user_id):
    status, payload = request("POST", f"{config['homeserver']}/_matrix/client/v3/createRoom", token_value, {
        "preset": "trusted_private_chat",
        "is_direct": True,
        "invite": [user_id],
        "name": "天气预报",
        "topic": "天气查询与每日自动推送",
    })
    if status not in (200, 201) or "room_id" not in payload:
        raise RuntimeError(f"create DM for {user_id} failed: HTTP {status} {payload}")
    return payload["room_id"]

def get_room(config, token_value, user_id):
    rooms = config.setdefault("direct_rooms", {})
    room_id = rooms.get(user_id)
    if room_id:
        return room_id
    room_id = create_dm(config, token_value, user_id)
    rooms[user_id] = room_id
    save_json(CONFIG_PATH, config)
    return room_id

def parse_hhmm(value):
    hour_text, minute_text = str(value).split(":", 1)
    hour = int(hour_text)
    minute = int(minute_text)
    if not (0 <= hour <= 23 and 0 <= minute <= 59):
        raise ValueError(f"invalid HH:MM value: {value}")
    return hour, minute

def normalized_hhmm(value):
    hour, minute = parse_hhmm(value)
    return f"{hour:02d}:{minute:02d}"

def sent_key(today, time_value):
    return f"{today} {normalized_hhmm(time_value)}"

def due_users(config, now=None):
    now = now or dt.datetime.now()
    today = now.date().isoformat()
    state = load_json(config["state_path"], {})
    sent = state.setdefault("last_sent", {})
    grace_minutes = int(config.get("schedule_grace_minutes", 15))
    users = []
    for user_id, settings in config.get("weather_users", {}).items():
        if not settings.get("enabled", True):
            continue
        time_value = settings.get("time", config.get("default_weather_time", "05:00"))
        try:
            hour, minute = parse_hhmm(time_value)
        except Exception as exc:
            print(f"skip {user_id}: invalid weather time {time_value!r}: {exc}", file=sys.stderr)
            continue
        if sent.get(user_id) == sent_key(today, time_value):
            continue
        scheduled = now.replace(hour=hour, minute=minute, second=0, microsecond=0)
        delta_seconds = (now - scheduled).total_seconds()
        if 0 <= delta_seconds < grace_minutes * 60:
            users.append(user_id)
    return users, state, today

def send_weather_to_users(config, user_ids, mark_sent=False, state=None, today=None):
    token_value = token(config)
    body = get_weather(config)
    sent = 0
    for user_id in user_ids:
        room_id = get_room(config, token_value, user_id)
        send_room(config, token_value, room_id, body)
        sent += 1
        if mark_sent and state is not None and today is not None:
            user_time = config.get("weather_users", {}).get(user_id, {}).get("time", config.get("default_weather_time", "05:00"))
            state.setdefault("last_sent", {})[user_id] = sent_key(today, user_time)
        time.sleep(0.2)
    if mark_sent and state is not None:
        save_json(config["state_path"], state)
    return sent

def main():
    config = load_json(CONFIG_PATH, None)
    if not config:
        raise SystemExit(f"missing config: {CONFIG_PATH}")
    if len(sys.argv) >= 2 and sys.argv[1] == "--send-now":
        users = sys.argv[2:] or list(config.get("weather_users", {}).keys())
        print(f"sent weather forecast to {send_weather_to_users(config, users)} room(s)")
        return
    users, state, today = due_users(config)
    if not users:
        print("weather schedule: no due users")
        return
    print(f"sent weather forecast to {send_weather_to_users(config, users, True, state, today)} room(s)")

if __name__ == "__main__":
    main()

七、配置天气调度与聊天同步

这一节把机器人变成真正可交互的联系人。天气推送使用 OpenRC 常驻服务，每 20 秒检查一次所有用户的设置时间；聊天同步也使用 OpenRC 常驻服务，每 20 秒处理一次邀请和命令。这样用户刚发完“设置天气时间 07:30”，到点后就由后台服务自动推送，不依赖 cron 正好踩中某一分钟。

#!/usr/bin/env python3
import datetime as dt
import hashlib
import html
import importlib.util
import json
import os
import re
import urllib.error
import urllib.parse
import urllib.request

CONFIG_PATH = "/root/data/docker_data/synapse/scripts/matrix-bot-weather.json"
WEATHER_SCRIPT_PATH = "/root/data/docker_data/synapse/scripts/matrix-bot-weather.py"
WELCOME = """你好，我是天气机器人。

你可以这样使用我：

天气
  立即查看今天和未来两天天气

设置天气时间 07:30
  设置每天自动推送时间

天气时间
  查看当前推送时间

关闭天气
  停止每日天气推送

开启天气
  恢复每日天气推送

帮助
  再次查看这份说明"""

def load_json(path, default):
    try:
        with open(path, "r", encoding="utf-8") as f:
            return json.load(f)
    except FileNotFoundError:
        return default

def save_json(path, data, mode=0o600):
    tmp = path + ".tmp"
    with open(tmp, "w", encoding="utf-8") as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
        f.write("\n")
    os.chmod(tmp, mode)
    os.replace(tmp, path)

def request(method, url, token=None, data=None, timeout=30):
    headers = {}
    body = None
    if token:
        headers["Authorization"] = f"Bearer {token}"
    if data is not None:
        body = json.dumps(data, ensure_ascii=False).encode("utf-8")
        headers["Content-Type"] = "application/json"
    req = urllib.request.Request(url, data=body, headers=headers, method=method)
    try:
        with urllib.request.urlopen(req, timeout=timeout) as resp:
            raw = resp.read().decode("utf-8", errors="replace")
            return resp.status, json.loads(raw) if raw else {}
    except urllib.error.HTTPError as e:
        raw = e.read().decode("utf-8", errors="replace")
        try:
            payload = json.loads(raw)
        except Exception:
            payload = {"raw": raw}
        return e.code, payload
    except Exception as e:
        return 0, {"error": str(e)}

def quote(value):
    return urllib.parse.quote(value, safe="")

def token(config):
    with open(config["token_path"], "r", encoding="utf-8") as f:
        return f.read().strip()

def local_user(config, user_id):
    return user_id.endswith(":" + config.get("server_name", ""))

def send_room(config, token_value, room_id, body):
    txn = "bot-reply-" + dt.datetime.now(dt.timezone.utc).strftime("%Y%m%dT%H%M%S")
    txn += "-" + hashlib.sha1((room_id + body).encode("utf-8")).hexdigest()[:10]
    html_body = "<br>".join(html.escape(line) for line in body.splitlines())
    content = {"msgtype": "m.text", "body": body, "format": "org.matrix.custom.html", "formatted_body": html_body}
    status, payload = request("PUT", f"{config['homeserver']}/_matrix/client/v3/rooms/{quote(room_id)}/send/m.room.message/{txn}", token_value, content)
    if status not in (200, 201):
        print(f"send failed {room_id}: HTTP {status} {payload}")

def enable_user(config, user_id, room_id=None):
    settings = config.setdefault("weather_users", {}).setdefault(user_id, {})
    settings.setdefault("enabled", True)
    settings.setdefault("time", config.get("default_weather_time", "05:00"))
    if room_id:
        config.setdefault("direct_rooms", {})[user_id] = room_id

def import_weather_module():
    spec = importlib.util.spec_from_file_location("matrix_bot_weather", WEATHER_SCRIPT_PATH)
    mod = importlib.util.module_from_spec(spec)
    spec.loader.exec_module(mod)
    return mod

def handle_command(config, token_value, room_id, sender, body):
    text = body.strip()
    lower = text.lower()
    enable_user(config, sender, room_id)
    if text in ("帮助", "help", "菜单"):
        send_room(config, token_value, room_id, WELCOME)
    elif text == "天气":
        weather = import_weather_module()
        send_room(config, token_value, room_id, weather.get_weather(config))
    elif text == "天气时间":
        s = config["weather_users"][sender]
        status = "已开启" if s.get("enabled", True) else "已关闭"
        send_room(config, token_value, room_id, f"你的天气推送{status}。\n当前时间：每天 {s.get('time', config.get('default_weather_time', '05:00'))}。\n城市：{config.get('city')}。")
    elif text == "关闭天气":
        config["weather_users"][sender]["enabled"] = False
        send_room(config, token_value, room_id, "已关闭每日天气推送。发送“开启天气”可以恢复。")
    elif text == "开启天气":
        config["weather_users"][sender]["enabled"] = True
        send_room(config, token_value, room_id, f"已开启每日天气推送。当前时间：每天 {config['weather_users'][sender].get('time', config.get('default_weather_time', '05:00'))}。")
    else:
        m = re.fullmatch(r"设置天气时间\s+([0-2]?\d):([0-5]\d)", text)
        if m:
            hour = int(m.group(1))
            minute = int(m.group(2))
            if hour > 23:
                send_room(config, token_value, room_id, "时间格式不对。示例：设置天气时间 07:30")
            else:
                config["weather_users"][sender]["time"] = f"{hour:02d}:{minute:02d}"
                config["weather_users"][sender]["enabled"] = True
                send_room(config, token_value, room_id, f"已设置成功：每天 {hour:02d}:{minute:02d} 给你推送天气预报。")
        elif lower in ("hi", "hello", "你好"):
            send_room(config, token_value, room_id, WELCOME)
        else:
            send_room(config, token_value, room_id, "我没看懂。可以发送“帮助”查看全部命令，或发送：设置天气时间 07:30")
    save_json(CONFIG_PATH, config)

def invite_sender(config, invite):
    events = invite.get("invite_state", {}).get("events", [])
    for event in events:
        if event.get("type") == "m.room.member" and event.get("state_key") == config.get("bot_user_id"):
            return event.get("sender")
    return None

def main():
    config = load_json(CONFIG_PATH, None)
    if not config:
        raise SystemExit(f"missing config: {CONFIG_PATH}")
    state = load_json(config.get("sync_state_path"), {})
    token_value = token(config)
    params = {"timeout": 0}
    if state.get("since"):
        params["since"] = state["since"]
    status, payload = request("GET", f"{config['homeserver']}/_matrix/client/v3/sync?{urllib.parse.urlencode(params)}", token_value, timeout=35)
    if status != 200:
        raise SystemExit(f"sync failed: HTTP {status} {payload}")
    for room_id, invite in payload.get("rooms", {}).get("invite", {}).items():
        sender = invite_sender(config, invite)
        if config.get("allow_local_users_only", True) and (not sender or not local_user(config, sender)):
            print(f"skip non-local invite {room_id} from {sender}")
            continue
        join_status, join_payload = request("POST", f"{config['homeserver']}/_matrix/client/v3/rooms/{quote(room_id)}/join", token_value, {})
        if join_status in (200, 201):
            enable_user(config, sender, room_id)
            send_room(config, token_value, room_id, WELCOME)
        else:
            print(f"join failed {room_id}: HTTP {join_status} {join_payload}")
    first_sync = not state.get("since")
    if not first_sync:
        for room_id, room in payload.get("rooms", {}).get("join", {}).items():
            for event in room.get("timeline", {}).get("events", []):
                if event.get("type") != "m.room.message":
                    continue
                sender = event.get("sender")
                if sender == config.get("bot_user_id") or not sender or not local_user(config, sender):
                    continue
                content = event.get("content", {})
                if content.get("msgtype") != "m.text":
                    continue
                handle_command(config, token_value, room_id, sender, content.get("body", ""))
    state["since"] = payload.get("next_batch", state.get("since"))
    save_json(config.get("sync_state_path"), state)
    save_json(CONFIG_PATH, config)
    print("sync ok")

if __name__ == "__main__":
    main()

八、增加服务器监控提醒

天气预报按用户自己的设置时间发送，监控提醒更适合“有事才吱声”。这里增加一个独立脚本，每 5 分钟检查一次：

  • 磁盘空间预警：检查 / 和 /var/lib/docker，默认使用率 80% 警告、90% 严重，或者可用空间低于 10GiB 警告。
  • Docker 容器异常通知：检查关键容器是否存在、是否运行、健康检查是否 unhealthy，以及重启次数是否增加。
  • TURN / LiveKit 状态检测：检查 coturn、LiveKit、LiveKit JWT、Element Call Web 相关容器、端口和 HTTP 探针。

核心逻辑建议按下面这个结构写：

#!/usr/bin/env python3
import datetime as dt
import hashlib
import html
import json
import os
import socket
import subprocess
import time
import urllib.parse
import urllib.request

CONFIG_PATH = "/root/data/docker_data/synapse/scripts/matrix-bot-monitor.json"

def load_json(path, default):
    try:
        with open(path, "r", encoding="utf-8") as f:
            return json.load(f)
    except FileNotFoundError:
        return default

def save_json(path, data):
    tmp = path + ".tmp"
    with open(tmp, "w", encoding="utf-8") as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
        f.write("\n")
    os.chmod(tmp, 0o600)
    os.replace(tmp, path)

def run(cmd):
    return subprocess.run(cmd, text=True, stdout=subprocess.PIPE, stderr=subprocess.PIPE, check=False)

def request(method, url, token=None, data=None, timeout=20):
    headers = {}
    body = None
    if token:
        headers["Authorization"] = f"Bearer {token}"
    if data is not None:
        body = json.dumps(data, ensure_ascii=False).encode("utf-8")
        headers["Content-Type"] = "application/json"
    req = urllib.request.Request(url, data=body, headers=headers, method=method)
    try:
        with urllib.request.urlopen(req, timeout=timeout) as resp:
            raw = resp.read().decode("utf-8", errors="replace")
            if not raw:
                return resp.status, {}
            try:
                return resp.status, json.loads(raw)
            except Exception:
                return resp.status, {"raw": raw[:200]}
    except Exception as e:
        return 0, {"error": str(e)}

def quote(value):
    return urllib.parse.quote(value, safe="")

def check_disks(config):
    issues = []
    disk = config["disk"]
    for mount in disk["mounts"]:
        proc = run(["df", "-P", "-B1", mount])
        if proc.returncode != 0:
            issues.append(f"[disk] {mount}: df failed")
            continue
        parts = proc.stdout.strip().splitlines()[-1].split()
        used_percent = int(parts[4].rstrip("%"))
        free_gb = int(parts[3]) / 1024 / 1024 / 1024
        if used_percent >= disk["critical_percent"]:
            issues.append(f"[disk:critical] {parts[5]} used={used_percent}% free={free_gb:.1f}GiB")
        elif used_percent >= disk["warn_percent"] or free_gb < disk["min_free_gb"]:
            issues.append(f"[disk:warning] {parts[5]} used={used_percent}% free={free_gb:.1f}GiB")
    return issues

def docker_inspect(names):
    result = {}
    for name in names:
        proc = run(["docker", "inspect", "--format", "{{json .}}", name])
        if proc.returncode != 0:
            result[name] = {"missing": True}
            continue
        data = json.loads(proc.stdout)
        state = data.get("State", {})
        health = state.get("Health") or {}
        result[name] = {
            "missing": False,
            "running": bool(state.get("Running")),
            "status": state.get("Status", "unknown"),
            "health": health.get("Status", "none"),
            "restart_count": int(data.get("RestartCount", 0)),
        }
    return result

def check_docker(config, state):
    issues = []
    previous = state.get("docker_restart_counts", {})
    current = {}
    for name, info in docker_inspect(config["docker"]["expected_containers"]).items():
        if info.get("missing"):
            issues.append(f"[docker] {name}: missing")
            continue
        current[name] = info["restart_count"]
        if not info["running"]:
            issues.append(f"[docker] {name}: not running, status={info['status']}")
        if info["health"] == "unhealthy":
            issues.append(f"[docker] {name}: health=unhealthy")
        old = previous.get(name)
        if old is not None and info["restart_count"] > old:
            issues.append(f"[docker] {name}: restart count {old} -> {info['restart_count']}")
    state["docker_restart_counts"] = current
    return issues

def tcp_open(port):
    try:
        with socket.create_connection(("127.0.0.1", int(port)), timeout=3):
            return True
    except Exception:
        return False

def udp_listening(port):
    proc = run(["ss", "-lun"])
    return proc.returncode == 0 and any(f":{port}" in line for line in proc.stdout.splitlines())

def check_realtime(config):
    issues = []
    rt = config["realtime"]
    for name, info in docker_inspect(rt["containers"]).items():
        if info.get("missing"):
            issues.append(f"[realtime] {name}: missing")
        elif not info["running"]:
            issues.append(f"[realtime] {name}: not running, status={info['status']}")
    for port in rt["tcp_ports"]:
        if not tcp_open(port):
            issues.append(f"[realtime] tcp/{port}: not listening")
    for port in rt["udp_ports"]:
        if not udp_listening(port):
            issues.append(f"[realtime] udp/{port}: not listening")
    for probe in rt["http_probes"]:
        status, payload = request("GET", probe["url"], timeout=5)
        if status not in probe["ok"]:
            issues.append(f"[realtime] {probe['name']}: HTTP {status} {payload}")
    return issues

def send_room(config, token, room_id, body):
    txn = "server-monitor-" + dt.datetime.now(dt.timezone.utc).strftime("%Y%m%dT%H%M%S")
    txn += "-" + hashlib.sha1((room_id + body).encode("utf-8")).hexdigest()[:10]
    html_body = "<br>".join(html.escape(line) for line in body.splitlines())
    content = {
        "msgtype": "m.text",
        "body": body,
        "format": "org.matrix.custom.html",
        "formatted_body": html_body,
    }
    url = f"{config['homeserver']}/_matrix/client/v3/rooms/{quote(room_id)}/send/m.room.message/{txn}"
    status, payload = request("PUT", url, token, content)
    if status not in (200, 201):
        raise RuntimeError(f"send failed: HTTP {status} {payload}")

def send_alert(config, token, state, body):
    rooms = set(config.get("target_rooms", []))
    weather_state = load_json(config.get("bot_weather_state_path", ""), {})
    for user_id, room_id in weather_state.get("direct_rooms", {}).items():
        if user_id in config["target_users"]:
            rooms.add(room_id)
    for room_id in sorted(rooms):
        send_room(config, token, room_id, body)
        time.sleep(0.2)

def main():
    config = load_json(CONFIG_PATH, None)
    if not config:
        raise SystemExit(f"missing config: {CONFIG_PATH}")
    state = load_json(config["state_path"], {})
    with open(config["token_path"], "r", encoding="utf-8") as f:
        token = f.read().strip()

    issues = sorted(set(check_disks(config) + check_docker(config, state) + check_realtime(config)))
    previous = set(state.get("active_issues", []))
    current = set(issues)
    new_issues = sorted(current - previous)
    recovered = sorted(previous - current)

    state["active_issues"] = issues
    state["last_check"] = dt.datetime.now().isoformat(timespec="seconds")

    if new_issues or recovered:
        lines = [f"服务器监控提醒 {dt.datetime.now().strftime('%Y-%m-%d %H:%M:%S')}", ""]
        if new_issues:
            lines.append("新增异常：")
            lines.extend(f"- {x}" for x in new_issues)
        if recovered:
            if new_issues:
                lines.append("")
            lines.append("已恢复：")
            lines.extend(f"- {x}" for x in recovered)
        send_alert(config, token, state, "\n".join(lines))

    save_json(config["state_path"], state)
    print(f"monitor ok: active={len(issues)} new={len(new_issues)} recovered={len(recovered)}")

if __name__ == "__main__":
    main()

十、聊天命令和用户自助设置

普通用户可以主动邀请机器人：

@bot:666666.xyz

机器人只自动接受本服用户邀请，也就是 :666666.xyz 结尾的用户。首次加入后会自动发送帮助说明：

你好，我是天气机器人。

你可以这样使用我：

天气
  立即查看今天和未来两天天气

设置天气时间 07:30
  设置每天自动推送时间

天气时间

  停止每日天气推送

  恢复每日天气推送

帮助
  再次查看这份说明

普通用户只会获得天气功能，不会收到服务器监控提醒。监控提醒由 matrix-bot-monitor.json 的 target_users 单独控制，例如只发给管理员：

"target_users": [
  "@honey:666666.xyz"
]

天气用户则保存在 matrix-bot-weather.json 的 weather_users 里，每个用户可以有自己的推送时间：

"weather_users": {
  "@honey:666666.xyz": {
    "enabled": true,
    "time": "05:00"
  },
  "@black:666666.xyz": {
    "enabled": true,
    "time": "07:30"
  }
}

一般不需要手动维护，用户直接邀请 @bot:666666.xyz 后会自动加入 weather_users。如果需要管理员直接改配置，可以编辑：

十二、关于加密房间

这个脚本使用的是普通 Matrix Client-Server API，没有实现端到端加密设备管理，也没有处理 Olm/Megolm 密钥。

因此建议让机器人使用未加密私聊或未加密房间。上面的脚本没有实现 E2EE，机器人即使能加入加密房间，也不能像正常客户端那样解密和处理消息。

如果一定要在端到端加密房间里使用机器人，需要换成支持 E2EE 的机器人框架，例如 matrix-nio[e2e]，并额外处理设备信任、密钥存储和恢复。

如果要立即发一次天气测试，可以直接运行：

/root/data/docker_data/synapse/scripts/matrix-bot-weather.py --send-now

如果要测试“定时是否真的生效”，不要只跑 --send-now。应该在 Matrix 客户端里把天气时间设置为下一分钟，然后观察 /var/log/matrix-bot-weather.log，确认是 matrix-bot-weather 服务自己到点发送。

天气脚本按每个用户配置的时间判断是否到点，并在 state 文件里按“日期 + 时间”记录已经发送，避免重复推送。聊天命令和天气推送是两条后台循环服务，监控提醒仍然是独立 cron；普通用户改天气时间不会影响监控提醒。
