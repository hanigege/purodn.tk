SMbox OPNsense 配置教程

在 OPNsense 中配置 SMbox DNS 下发、网关、别名、NAT 转发和 LAN 防火墙规则。

开始前先确认两个地址：

  • 10.0.0.2 需要替换为你自己的 SMbox IP。
  • 10.0.0.1 需要替换为你自己的 OPNsense IP。

修改 DNS 下发为主路由

新建网关配置

按下图新建一个网关配置：

打开防火墙别名

打开 防火墙 -> 别名，新建两个别名：proxy_cidr 和 proxy_device。

proxy_cidr 为需要代理的 IP 地址：

198.18.0.0/15
91.108.56.0/22
91.108.4.0/22
91.108.8.0/22
91.108.16.0/22
91.108.12.0/22
149.154.160.0/20
91.105.192.0/23
91.108.20.0/22
185.76.151.0/24

proxy_device 为需要代理设备的 MAC 地址。

任何时候不要把 SMbox 的 IP 加入到 proxy_device 中，否则会引起环回导致崩溃。
