Hermes TUI 终端 CLI 命令速查

Hermes 终端 TUI 下所有核心快捷键、斜杠命令和使用技巧，覆盖日常聊天、代码审查、计划模式、会话管理等完整操作。

什么是 Hermes TUI

Hermes 的 TUI（终端用户界面）让你直接在终端中和 AI 交互，不需要浏览器或额外的 GUI。启动方式：

# 直接用 hermes 命令进入 TUI（如果配置了默认 profile）
hermes

# 或指定 profile
hermes chat --profile code

# 或者使用单独的子命令
hermes tui

进入 TUI 后，你会看到类似聊天软件的界面——上方是对话历史，底部是输入框。在输入框里，你可以随时按 Tab 看到所有可用命令的自动补全列表。

核心特性：

  • 全终端操作，无需鼠标
  • 自动补全覆盖所有命令和已安装 skill
  • 计划模式安全预览再执行
  • 会话管理方便回溯历史

常用快捷键

快捷键作用
/调出斜杠命令自动补全下拉菜单
Tab接受自动建议 / 补全斜杠命令
↑ / ↓浏览历史消息（按方向键循环）
Ctrl+C中断当前 AI 输出
Ctrl+D退出 TUI
Ctrl+L清屏（保留会话历史）
Ctrl+U清空当前输入行
Ctrl+W删除光标前一个单词

提示：按 / 后会自动弹出补全菜单，显示所有可用的斜杠命令。如果装了多个 skill，菜单会很长——直接输入关键字过滤即可。

核心斜杠命令

命令作用
/plan计划模式。只做规划和分析，不执行任何代码/操作，生成 markdown 计划保存在 ~/.hermes/plans/ 目录。等你确认后才动手
/review代码审查。对当前对话中的代码进行自动审查；也可主动提交代码让我审查
/model显示当前使用的模型，或切换模型（会列出可用模型列表供选择）
/new新会话——清空对话历史，从头开始
/undo回退最近 N 条用户消息重做。不加数字则回退 1 条。例如 /undo 3 回退 3 轮
/retry重试上一条 AI 回复，重新生成
/clear清屏 + 新会话，比 /new 多一个清屏动作
/compress压缩上下文节省 Token。加数字保留最近 N 轮对话，如 /compress 20 保留最近 20 轮
/yolo切换 YOLO 模式。开启后会跳过所有危险命令审批，直接执行——请谨慎使用
/verbose切换工具输出显示级别。循环切换：关 → 新 → 全部 → 详细
/goal设置一个持续目标。目标会跨回合保持，直到你完成或取消它。例如 /goal 完成用户注册模块
/stop杀掉所有后台进程（长时间运行的任务、构建、测试等）
/title给当前会话设置标题。方便后续通过 session_search 查找定位
/session浏览和恢复历史会话。列出所有历史会话，可选择恢复或删除

斜杠命令使用技巧

  • 大部分命令支持部分匹配，比如 /pla 也能触发 /plan
  • 带数字参数的命令（/undo 3、/compress 20）如果不传数字则使用默认值
  • /yolo 是开关，再输一次会关掉 YOLO 模式
  • /verbose 是四级循环，输一次切一级

计划模式详解

通过 /plan 触发后，Hermes 进入只分析不执行模式：

  1. 我会先分析你的需求、列出可能的方案、评估风险和依赖
  2. 输出一份 markdown 计划到 ~/.hermes/plans/ 目录
  3. 不会执行任何命令、修改任何文件
  4. 你确认后说"动手"或"执行"，我再开始操作

适用于：

  • 不确定影响范围的改动，先看计划再决定
  • 复杂的多步骤操作，需要分步确认
  • 学习场景——只看分析和方案，不执行

退出计划模式：按 Ctrl+C 取消当前计划，或执行完你同意的操作后自动退出。

用 /goal 实现跨会话目标

/goal 是持续目标系统，适合拆分大任务：

# 设置目标
/goal 重构用户认证模块

# 之后每次对话，Hermes 都会记住这个目标
# 完成一部分后继续对话，会基于上下文推进
# 目标完成后手动取消
/goal 重构用户认证模块 --done

目标会持久化——即使中间做其他任务、开关 TUI，目标也不会丢失。

Skill 自动注册斜杠命令

所有已安装的 skill 会自动变成斜杠命令。例如装了 github-workflows skill，直接输入 /github-workflows 就能触发对应的 workflow。

查看已安装的 skill：

hermes skills list

每个 skill 内置了专门的流程指导——触发后 Hermes 会加载该 skill 的步骤、参考资料和校验检查清单。

建议：在终端里敲 / 看看自动补全菜单，所有可用命令一目了然😊

环境变量与配置

TUI 模式也受以下环境变量影响：

变量作用
HERMES_PROFILE指定使用的 profile（默认 default）
HERMES_HOME自定义 Hermes 配置目录
HERMES_LOG_LEVEL日志级别（debug/info/warn/error）

也可以在启动时指定：

HERMES_PROFILE=code hermes

总结

Hermes TUI 把整个 AI 助手功能搬到了终端里，用键盘就能完成一切操作。上面这些快捷键和斜杠命令覆盖了日常 95% 的使用场景：

  • 快捷键：控制界面和中断
  • 斜杠命令：控制会话、模型、模式、审查等核心功能
  • 计划模式：安全预览再执行
  • 持续目标：跨会话推进大任务
  • Skill 系统：按需扩展能力

在终端里敲 / 看看自动补全菜单——比死记硬背快得多。
